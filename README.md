1. Prérequis:

- Docker (processus lancé)

2. Installation:

    1. Le nom du conteneur qui sera généré prendra le nom du dossier racine du projet (ex: "monsite"), renommez le si besoin.
    2. Se rendre dans le dossier racine du projet via le terminal, executer :

        - docker-compose build
        - docker-compose up -d

    3. Se rendre sur https://localhost (rafraichir 2 fois la page si message d'erreur d'accès à la DB)

3. Dépendances:
    phpmailer
    twig
